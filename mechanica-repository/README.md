# Mechanica Repository

Dit is een startpunt voor een nieuwe repository over **mechanica**.

## Onderwerpen
- Kinematica
- Dynamica
- Krachten en momenten
- Arbeid, energie en vermogen
- Trillingen en golven

## Structuur (voorstel)
```
mechanica-repository/
├── README.md
├── docs/
│   ├── theorie/
│   └── oefeningen/
├── src/
│   ├── simulaties/
│   └── tools/
└── assets/
    ├── afbeeldingen/
    └── data/
```

## Volgende stappen
1. Maak een nieuwe GitHub-repository aan met naam `mechanica-repository`.
2. Kopieer de inhoud van deze map naar de nieuwe repository.
3. Voeg eerste lesmateriaal en oefeningen toe in `docs/`.
